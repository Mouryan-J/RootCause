# PM-058 — Cloudflare

**Company:** Cloudflare  
**Category:** Hardware/Power Failures  
**Source:** https://web.archive.org/web/20211015231917/https://blog.cloudflare.com/a-byzantine-failure-in-the-real-world/

## Incident Summary

A partial switch misbehavior caused a cascading Byzantine failure which impacted the availability of the API and dashboard for six hours and 33 minutes.
